El proyecto

¿Qué es Archivo Cero?

Es una colección de casos digitales diseñados como experiencias de investigación inmersiva. Accedés a archivos recuperados y organizados para que reconstruyas lo ocurrido — desde tu navegador, sin instalar nada.

El punto de partida

Cada caso comienza con un Archivo 0 que introduce el contexto y explica cómo interactuar con el material. A partir de ahí, la experiencia avanza de forma lógica y progresiva.
